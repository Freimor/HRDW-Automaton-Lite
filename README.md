## HRDW-Automaton-Lite
RP2040 BadUSB. Updated design for easier soldering with stencil and heat plate.
- Robust Micro USB connector
- All soldering on the top side
- ESD protection
- Simple construction with 3d printed case

The project is completed in Q1 2025.
